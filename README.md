##  Ekranlar

- **Home Screen:** Rastgele yemek tariflerin gösterildiği sayfa
- **Favorites Screen:** Kaydedilmiş favori tariflerin olduğu sayfa  
- **Detail Page:** Yemeğin detayları, malzemeler ve açıklamalarının olduğu sayfa  

---

## Kullanılan Teknolojiler

- React Native
- Expo
- React Navigation

---

##  Kurulum ve Çalıştırma

### 1) Bağımlılıkları yükle
```
npm install
```

### 2) Projeyi başlat
```
npx expo start
```
